# react-native-template

Full typescript support

## Install

run `yarn` or `npm install`

### IOS

run `yarn pod` && `yarn ios`

### Android 

run `yarn android`

## Rename project

run `yarn rename newProjectName`

## GraphQL server

- Install graphql server

run `yarn server:install`

- Run server

run `yarn server:start`
